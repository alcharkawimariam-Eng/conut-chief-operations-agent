# 📖 Data Dictionary: Conut Bakery Operational Dataset

This document defines the schema for the cleaned datasets located in the `/data/prepared data/` directory. All raw report artifacts (headers, page markers, and formatting noise) have been removed, and numeric values have been normalized for calculation.

---

## 1. `cleaned_sales_detail.csv`
*Market Basket Analysis, Combo Identification, and Modifier Tracking.*

| Column | Description | Data Type |
| :--- | :--- | :--- |
| **Branch** | The store location where the transaction occurred. | String |
| **Customer** | Anonymized unique identifier for the customer (e.g., Person_0129). | String |
| **Qty** | The quantity of the specific item sold. | Float |
| **Item** | The product name or modifier (e.g., [PISTACHIO MILKSHAKE]). | String |
| **Price** | The unit price or total line-item value. | Float |

---

## 2. `cleaned_customer_orders.csv`
*Customer Lifetime Value (CLV) and Retention Analysis.*

| Column | Description | Data Type |
| :--- | :--- | :--- |
| **Customer_Name**| Anonymized ID (Primary Key for customer joins). | String |
| **Phone** | Anonymized phone contact identifier. | String |
| **First_Order** | Date and time of the customer's first visit. | DateTime |
| **Last_Order** | Date and time of the customer's most recent visit. | DateTime |
| **Total_Spent** | Total lifetime revenue generated by this customer. | Float |
| **Order_Count** | Total number of unique orders placed by this customer. | Integer |

---

## 3. `cleaned_sales_by_item_group.csv`
*Menu Engineering and Category Performance (Coffee vs. Food).*

| Column | Description | Data Type |
| :--- | :--- | :--- |
| **Branch** | The store location. | String |
| **Division** | Top-level category (e.g., Hot-Coffee Based). | String |
| **Group** | Sub-category grouping for the items. | String |
| **Item** | The specific product name (e.g., AMERICAN COFFEE). | String |
| **Qty** | Total units sold during the reporting period. | Float |
| **Total_Amount** | Aggregate revenue generated by this item. | Float |

---

## 4. `cleaned_branch_tax_summary.csv`
*High-level Branch Benchmarking and Tax Audit.*

| Column | Description | Data Type |
| :--- | :--- | :--- |
| **Branch** | The store location. | String |
| **VAT_11_Percent**| The calculated 11% Value Added Tax for the branch. | Float |
| **Total_Revenue** | The total gross revenue recorded for the location. | Float |

---

## 5. `cleaned_monthly_branch_sales.csv`
*Demand Forecasting, Seasonality, and Trend Tracking.*

| Column | Description | Data Type |
| :--- | :--- | :--- |
| **Branch** | The store location. | String |
| **Month** | Name of the month. | String |
| **Year** | Calendar year of the sales. | Integer |
| **Total_Sales** | Total revenue recorded for that specific month. | Float |
| **Date** | A normalized timestamp (YYYY-MM-01) for time-series analysis. | Date |

---

## 6. `cleaned_avg_sales_menu.csv`
*Channel Performance and Ticket Average Analysis.*

| Column | Description | Data Type |
| :--- | :--- | :--- |
| **Branch** | The store location. | String |
| **Menu_Type** | Service channel: DELIVERY, TABLE, or TAKE AWAY. | String |
| **Customer_Count**| Number of unique orders/customers in this channel. | Float |
| **Total_Sales** | Total revenue generated via this channel. | Float |
| **Avg_Spent_Per_Customer** | The "Ticket Average" ($Total Sales / Customer Count$). | Float |

---

## 7. `cleaned_attendance_logs.csv`
*Labor Efficiency and Staffing Cost Estimation.*

| Column | Description | Data Type |
| :--- | :--- | :--- |
| **Branch** | The location where the shift was performed. | String |
| **Employee_ID** | Unique ID for the staff member. | String |
| **Employee_Name** | Anonymized name of the staff member (e.g., Person_0001). | String |
| **Date** | Date of the shift worked. | Date |
| **Punch_In** | Clock-in time (HH.MM.SS). | String |
| **Punch_Out** | Clock-out time (HH.MM.SS). | String |
| **Work_Hours** | Shift duration converted to **decimal hours** (e.g., 11.97). | Float |

---

## 8. `division_summary_clean.csv`
*Multi-channel Category Analysis.*

| Column | Description | Data Type |
| :--- | :--- | :--- |
| **Branch** | The store location. | String |
| **Division** | Product category (e.g., CHIMNEY TOPPINGS). | String |
| **Delivery** | Revenue from this division via Delivery. | Float |
| **Table** | Revenue from this division via Dine-In. | Float |
| **TakeAway** | Revenue from this division via Take-away. | Float |
| **Total** | Total revenue for this division across all channels. | Float |
